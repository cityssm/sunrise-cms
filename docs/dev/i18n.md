# How to Use i18next in Different Contexts

In all cases, the key is:

```text
{nameSpaceFileName}:{jsonKey}
```

## EJS

```ejs
<h1 class="title">
  <%= t('contracts:funeralHomes') %>
</h1>
```

## Client Side Javascript

```typescript
import type { i18n } from 'i18next'

declare const i18next: i18n

document
  .querySelector('#cityssm-theme--logout-button')
  ?.addEventListener('click', (clickEvent) => {
    clickEvent.preventDefault()

    bulmaJS.confirm({
      contextualColorName: 'warning',
      title: i18next.t('login:logout'),

      message: i18next.t('login:logoutConfirm'),

      okButton: {
        callbackFunction: doLogout,
        text: i18next.t('login:logout')
      }
    })
  })
```

## HTML Loaded by Javascript (i.e. Static HTML Modals)

This will localize the selected element, and every element beneath it.

Put the key in the `data-i18n` attribute.
By default, the `textContent` for the element will be populated.
To populate an attribute, include the attribute name in `data-i18n-attribute`.

```html
<div
  class="modal"
  data-i18n="admin:addUser"
  data-i18n-attribute="aria-label"
  aria-label="Add User"
></div>
```

```html
<label class="label" data-i18n="admin:userName" for="userName">User Name</label>
```

```typescript
sunrise.localize(modalElement)
```
